# Recommended next steps:

### Note:

1. Make sure you have installed native code dependencies [here](https://reactnative.dev/docs/environment-setup#installing-dependencies)
2. Make sure your `node version` is minimum `23`.

### To preview and run the project on your device:

1. Open project folder in <u>Visual Studio Code</u>
2. Run `npm install` in the terminal
3. Run `npx expo start` in the terminal
4. Run on For iOS device (only on MacOS)
   1. Press `i` to view on iOS simulator or follow the instructions [here](https://docs.expo.dev/workflow/run-on-device/) to run on a physical device.
5. Run on For android device
   1. Press `a` to view on Android Virtual Device or follow the instructions [here](https://docs.expo.dev/workflow/run-on-device/) to run on a physical device.

# TechnaVeia

Projeto desenvolvido para a disciplina de **Desenvolvimento Mobile I**, com o objetivo de criar soluções digitais voltadas para serviços de tecnologia e suporte em informática.

---

## 📌 Sobre o projeto

A TechnaVeia é uma proposta de plataforma que conecta usuários a serviços de tecnologia, como manutenção de computadores, suporte técnico e soluções digitais.

Além disso, foi desenvolvida uma aplicação CLI (Command Line Interface) utilizando **TypeScript**, com persistência de dados em **SQLite** e manipulação através do **Drizzle ORM**, implementando um sistema com CRUD completo.

---

## ⚙️ Funcionalidades da aplicação CLI

- Cadastro, listagem, atualização e remoção de:
  - UF
  - Cidade
  - Região
- Relacionamento entre as entidades
- Persistência de dados com SQLite
- Listagem completa no formato:

## 📌 Exemplo de listagem


DF-Brasília - Asa Norte
SP-São Paulo - Praça da Sé


---

## 🛠️ Tecnologias utilizadas

 TypeScript  
- Node.js  
- SQLite  
- Drizzle ORM
